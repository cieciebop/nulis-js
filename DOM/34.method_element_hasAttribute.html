Bab Method element hasAttribute

Method hasAttribute() digunakan untuk memeriksa apakah sebuah atribut sudah di set di dalam element HTML atau belum. Atribut yang akan dicek diinput sebagai argumen dari method ini. Hasil akhir dari method hasAttribute() adalah boolean true atau false. True jika atribut tersebut ditemukan, dan false jika atribut itu tidak ditulis.
